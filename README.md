# 📊 Rare-Earth Thermodynamics Plotter

A Python visualization tool built to analyze thermodynamic trends in rare-earth liquid metals (La, Ce, Pr, Eu, Yb).

### 🔍 Scientific Context
This script visualizes the inverse relationship between **Isothermal Compressibility ($\beta_T$)** and **Sound Velocity ($\vartheta_s$)** at the melting temperature. 
Data is derived from Mean Spherical Model Approximation (MSMA) calculations solved via Fortran.

### 🛠️ Libraries Used
- `matplotlib`
- `numpy`
